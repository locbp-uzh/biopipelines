This session is run on a local computer, not on the cluster, so folders and files are not accessible, the environments are not set, and no tool requiring a gpu can run.
Remember that tools shouldn't write python code, only bash code, and python code meant to run at execution time should be in HelpScripts (pipe_ scripts for tools), and shouldn't write bash code.
When implementing new code or refactoring, never introduce unnecessary fallbacks or guessed default values.
